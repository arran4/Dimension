## Dimension

Dimension is a Beer to Peer transfer service.

### Project Status

This project is currently being ported from C# to a Flutter/Dart application.

Currently, all the original C# code has been moved into `.dart` files within the `lib/` directory, enclosed in comment blocks. The next steps involve migrating these structures and logic into proper Dart code.
